# 要素化民事起诉状模板 · 物业服务合同纠纷

> 与 API 契约 v1.0.0（已冻结 2026-06-24）对齐；契约为派生 SSOT，见 `docs/api/`。
> 配套：[M2 项目批次案件主数据](./02-项目批次案件主数据.md)（要素采集）、[M4 催收作业](./04-催收作业.md) BR-M4-18（平台用要素化数据套模板出具 PDF）、[M6 存证](./06-存证.md) BR-M6-08（证据包）。
> 用途：平台「生成诉讼文件」时，用**项目（小区）+ 案件 + 催收过程**的要素化数据自动套入下方模板生成 PDF；要素缺失则在预览中标红、提示补全。原型已接入「案件详情 → 生成诉讼文件（要素预览）」。

---

## 1. 模板正文（占位符 `{}` 为要素槽位）

```
民事起诉状

原告：{物业公司名称}
　　　统一社会信用代码：{统一社会信用代码}
　　　住所地：{物业/小区地址}
　　　法定代表人：{法定代表人}，联系电话：{原告联系电话}

被告：{业主姓名}
　　　身份证号：{业主身份证号}
　　　房屋坐落：{小区名称}{房号}
　　　联系电话：{业主联系电话}

诉讼请求：
一、判令被告支付拖欠物业服务费 {欠费金额} 元（欠费周期：{欠费周期}）；
二、判令被告支付违约金（{违约金规则}，计算至实际清偿之日止）；
三、本案诉讼费由被告承担。

事实与理由：
原告依据《{合同类型}》为 {小区名称} 提供物业服务，
收费标准为 {物业费标准}，收费周期为 {收费周期}。
被告系上述房屋业主，自欠费周期起拖欠物业服务费共计 {欠费金额} 元，
原告经多次催告（见催收时间线、通话录音及转写、送达签收存证）仍未获支付，
依《中华人民共和国民法典》物业服务合同相关规定，被告应承担给付义务及违约责任。

证据清单：
1.《{合同类型}》及附件（证明合同关系与收费标准）；
2. 收费标准 / 业主大会决议（证明收费依据）；
3. 催收时间线 + 通话录音及转写 + 送达签收存证（证明催告与送达，M4/M6 存证打包，BR-M6-08）；
4. 房屋坐落及业主信息（证明被告主体）。

此致
{受理法院}

　　　　　　　　　　　　　　　　具状人（盖章）：{物业公司名称}
　　　　　　　　　　　　　　　　　　　　　　　　{起诉日期}
```

## 2. 要素来源映射（哪个模块提供）

| # | 要素槽位 | 数据来源 | 采集环节 | 必填 |
|---|---|---|---|---|
| 1 | 物业公司名称（原告） | 项目（小区） | 新建/编辑项目 | ✅ |
| 2 | 统一社会信用代码 | 项目（小区） | 新建/编辑项目 | 建议 |
| 3 | 物业/小区地址 | 项目（小区） | 新建/编辑项目 | ✅ |
| 4 | 法定代表人 | 项目（小区） | 新建/编辑项目 | ✅ |
| 5 | 原告联系电话 | 项目（小区） | 新建/编辑项目 | 建议 |
| 6 | 合同类型 | 项目（小区） | 新建/编辑项目 | ✅ |
| 7 | 物业费标准 | 项目（小区） | 新建/编辑项目（按业态可多行） | ✅ |
| 8 | 收费周期 | 项目（小区） | 新建/编辑项目 | 建议 |
| 9 | 违约金规则 | 项目（小区） | 新建/编辑项目 / 减免政策 | ✅ |
| 10 | 业主姓名（被告） | 案件 | 批次导入采集 | ✅ |
| 11 | 业主身份证号 | 案件 | 批次导入采集（同业主识别口径，BR-M4-21） | 建议 |
| 12 | 房屋坐落（小区+房号） | 项目 + 案件 | 项目名 + 案件房号 | ✅ |
| 13 | 业主联系电话 | 案件 | 批次导入采集 | 建议 |
| 14 | 欠费金额 | 案件 | 应收（导入）| ✅ |
| 15 | 欠费周期 | 案件 | 批次导入采集 | ✅ |
| 16 | 证据包（合同/录音/送达存证） | M4 / M6 | 催收过程 + 存证打包（BR-M6-08） | — |
| 17 | 受理法院 / 起诉日期 | 出具时填写 | 平台出具环节 | — |

**结论**：项目（小区）档案承载"原告 + 合同 + 收费 + 违约金"要素，案件承载"被告 + 欠费"要素，M4/M6 承载"证据"要素，三者拼成完整起诉状。任一必填要素缺失，原型预览会标红并阻止生成正式 PDF。

## 3. 与现有规则的关系

- 平台**用要素化数据套模板自动生成 PDF（可打印）**（BR-M4-18 / US-M4-09）；法律服务**按次付费**（M9）。
- 起诉要素**随案件导入采集**（M2，案件主数据「起诉要素字段集」），项目级要素在**项目档案**维护（BR-M2-11）。
- 证据材料打包按 **ZIP**：催收时间线 PDF + 过程文件（录音/转写/工单/送达存证），可独立核验（BR-M6-08 / BR-M6-04）。

> 注：本模板为**演示与字段对齐用**的通用范式，正式诉状文本以各地法院要素式起诉状格式与执业律师审核为准。
