## node --check 方法（任务 2/3 通用）

`index.html` 是 HTML，`node --check` 不能直接吃。用以下方式只校验内联脚本语法：
```bash
# 抽出 <script> 内 JS 到临时文件再 check（按项目实际 script 标签调整）
node -e "const fs=require('fs');const h=fs.readFileSync('/Users/shuo/AI/cuishou/docs/ui/高保真/index.html','utf8');const m=[...h.matchAll(/<script[^>]*>([\s\S]*?)<\/script>/g)].map(x=>x[1]).filter(s=>!/src=/.test(s)).join('\n;\n');fs.writeFileSync('/tmp/_chk.js',m);"
node --check /tmp/_chk.js && echo OK
```
> 若内联脚本用了 ESM/浏览器全局（Vue、document），`--check` 只验语法不验运行，足够防括号/逗号错。改完务必跑一次。
