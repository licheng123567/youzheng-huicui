# 草图 · 01 核心闭环线框（低保真）

> 覆盖核心主路径（★核心页）。低保真线框只定**布局、信息分区、操作位置**，不定视觉。
> 角色差异统一用 `〔角色〕` 标注按钮的可见角色（见 `02-角色操作权限矩阵.md`）。
> 角色代号：SA 平台超管 / SE 平台员工 / PL 物业负责人 / PC 物业协调员 / VL 服务商负责人 / CO 催收员。
