## 1. PC 后台 `index.html`（6 角色 + 角色切换器）

### 1.1 跨切面护城河（最高优先）
- [ ] **资金双线隔离**：物业侧只见收佣线、服务商侧只见付佣线，互不可见
- [ ] **三方数据隔离**：物业看本物业、服务商看本商、平台看全局；跨组织一律只读/不可见
- [ ] **结案脱敏**：结案案件业主敏感信息收敛，仅留统计（BR-M8-09）
- [ ] **SE 数据范围**：平台员工按 dataRange 过滤（案件/报表/审计/质检不越界）
- [ ] **权限矩阵**：参数配置「权限矩阵 14×6」与各页实际显隐一致（●◐○▲）

### 1.2 核心闭环
- [ ] 工作台今日必办可点直达；管理角色（PL/VL/SA）经营概览（今日/本月/自定义）+ 回款趋势
- [ ] 项目 → 批次 → 案件三级；案件详情三栏（画像/时间线·项目资料·作战手册/操作区）
- [ ] 撮合派单（右侧抽屉，整批/拆单 + 容量护栏 BR-M3-23）、全局/服务商公海、T2 退回
- [ ] 减免阶梯·决定权：催收员自决直接生效 / 线下内部流程仅留痕（**无伪审批待办**）
- [ ] 坏账/撤案：填原因留痕、**不设审核流**

### 1.3 结算与计费
- [ ] 收佣对账（物业只查看、平台确认收款 BR-M9-12b）；案件级异议→平台调明细重算（BR-M9-12c）
- [ ] 催收员佣金：人→批次（比例可不同）→明细→勾选生成支付单据→结算单列表确认支付=结算完成
- [ ] 计费明细 月→日→明细下钻（仅用量不金额）；能力用量汇总只用量（US-M10-02）
- [ ] 充值后台充值（不开放自助）；短信统计+明细合并、时段筛选、导出

### 1.4 平台 AI 后台（SA，SE 需授权）
- [ ] 模型与密钥：DeepSeek LLM + 百炼 paraformer-8k-v2（密文）
- [ ] 通话前/后提示词；**通话中实时提示恒关闭**
- [ ] 作战手册飞轮：分级采纳闸（强制人工 / 低风险自动可回滚 / 全人工），**人工采纳闸**
- [ ] 话术库：AI + 专家两来源、效果查询（使用次数/承诺·回款转化）、自我迭代变体 A/B + 晋升/驳回
- [ ] 质检复核：确认→建处置任务 / 误报撤销 / 升级；风险处置任务跟踪（平台监管，两侧不可见）

### 1.5 组织与成员
- [ ] 新建组织（右侧抽屉）；平台**只建管平台员工**，跨组织成员只读
- [ ] 角色模板/权限子集；改绑手机 + 重置密码交接同一账号

## 2. 催收员 / 协调员 App `app.html`
- [ ] 顶部切 催收员(CO)/协调员(PC)；底部 4 Tab（工作台/案件/通话记录/我的）
- [ ] **工作台本日概览**：通话分钟 / 通话数量 / 回款金额 / 待处理数
- [ ] **拨打**：跳起本机拨号器、系统录音；平台不主动拨号、**无通话中实时提示**（BR-M4-01b）
- [ ] **获取最新通话录音**：拉取中 → 已就绪（时间/时长/号码）→ 解析

- [ ] **救济**：拉不到 → 调整目录 / Web 手动上传（同一解析链路）
- [ ] CO/PC 均可拨打/获取录音（BR-M4-01a）
- [ ] **协调员**：标记线下回款（金额/日期/渠道/备注→触发结清判定）、处理工单（回执互推闭环）、法务送达签收拍照
