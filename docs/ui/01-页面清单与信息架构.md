## 3. 各角色导航树（左导航按权限过滤）

> 同一套菜单字典，按角色权限点过滤渲染。`—` 表示该角色无此入口。

### 平台超管
```
工作台
组织与权限 ▸ 组织管理 / 成员授权 / 角色模板 / 操作日志
撮合派单   ▸ 派单工作台 / 平台公海
项目数据   ▸ 项目 / 批次 / 案件（全平台）
对账结算   ▸ 收佣对账 / 付佣对账 / 充值 / 计费明细
报表       ▸ 平台损益 / 物业经营 / 服务商经营
系统设置   ▸ 参数配置(CFG) / 短信通道 / 存证通道 / 话术库
```

### 平台员工（受数据范围 区域/服务商/物业 限制）
```
工作台 / 撮合派单(若授权) / 项目数据(范围内) / 对账(若授权) / 报表(范围内)
※ 菜单与超管同构，按授予的功能权限点 + 数据范围裁剪
```

### 物业负责人
```
工作台
项目管理 ▸ 项目档案 / 减免规则 / 批次 / 案件
法务存证 ▸ （查看，主导在协调员） — 视权限
对账     ▸ 收佣对账（仅平台↔本物业）/ 充值（分钟+短信）
报表     ▸ 物业经营报表
成员     ▸ 协调员管理
```

### 物业协调员
```
工作台
案件     ▸ 项目/批次/案件（本物业）
催收协作 ▸ 线下回款标注 / 工单回执 / 缴费链接
法务存证 ▸ 法务工作区（催收函/律师函/起诉状）/ 存证管理 / 送达签收
结案     ▸ 撤案/坏账（须选原因）
```

### 服务商负责人
```
工作台
案件     ▸ 服务商公海 / 全部案件（本服务商）
分配     ▸ 案件分配（整批/拆/平均分）
对账     ▸ 付佣对账（仅平台↔本服务商）/ 充值（分钟）
报表     ▸ 服务商经营报表
成员     ▸ 催收员管理
```

### 催收员
```
工作台（承诺到期/待办）
我的案件（私海）
服务商公海（抢单）
开放抢单（跨服务商可见的开放案件）
```

### 业主（H5，无登录，无导航）
```
单页：账单明细 + 收款信息
```

## 4. 路由草案（高保真原型用）

```
/login                         登录
/                              工作台（按角色 mock）
/projects                      项目列表
/projects/:id                  项目档案
/batches                       批次列表
/batches/:id                   批次详情
/batches/import                批次导入向导
/cases                         案件列表
/cases/:id                     案件详情（统一全集页）★复用核心
/dispatch                      撮合派单工作台
/sea/platform                  平台公海
/sea/provider                  服务商公海
/cases/mine                    我的案件（私海）
/work/desk                     催收员工作台
/cases/:id/call/:callId        通话&转写详情
/legal/:caseId                 法务工作区
/recon/in                      收佣对账
/recon/out                     付佣对账
/recharge                      充值中心
/reports/platform              平台损益报表
/settings/cfg                  参数配置
```

> 原型以 hash 路由或多 HTML 文件实现，零构建可双击打开。角色切换器改写全局 `currentRole` 后重渲染当前页。
